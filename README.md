# Lector CBZ
Lector local de archivos `.cbz` ejecutado completamente en el navegador.

Permite:

- Leer archivos CBZ.
- Cargar carpetas completas.
- Buscar por nombre, autor o tags.
- Leer en modo paginado o cascada.
- Exportar obras a PDF.
- Procesar archivos localmente sin subirlos a servidores.

---

## Uso

### Opción 1 — Usar en línea

Abrir la página web del proyecto y cargar los archivos `.cbz` desde el navegador.

Los archivos:

- no se suben al servidor;
- no se almacenan remotamente;
- se procesan únicamente en el navegador del usuario.

---

### Opción 2 — Ejecutar localmente

#### Requisitos

- Navegador moderno:
  - Brave (probado)
  - Chrome
  - Edge
  - Firefox
- Visual Studio Code
- Extensión Live Server

---
